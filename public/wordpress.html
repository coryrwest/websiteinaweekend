Installing WordPress

Please make sure you have received an email from HostGator about your hosting before continuing.

We will be using WordPress as the platform for your new website. WordPress is a very popular, secure, easy to use platform for building and maintaining websites. Because of its wide-spread use there are tons of tutorials online about how to use it. For the purposes of this guide, we will teach you everything you need to know.

If you are using HostGator hosting

If you are using HostGator hosting then refer to the steps below.

In your HostGator control panel there should be a link in the Software/Services section called Quick Install (see below). Click that link and you will be taken to the Quick Install installer.

Next, click on WordPress from the menu on the left.

Next, click Continue.

Next, click fill in the required information. We will be installing WordPress in the root directory of your hosting, so leave the second box blank (the one with websiteinaweekend in it in the below screenshot). Make sure that Enable Auto Upgrades is unchecked. Then click Install Now.

The installer will install WordPress on your hosting using the details you provided. When it is finished it will give you a link and a username and password. Make sure you write these down as we will need them later in the guide.

Now that WordPress is installed and set up we can log in and work on filling it with content.
